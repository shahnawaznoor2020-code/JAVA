## Java Practice Question – Volume of a Sphere

This folder contains a Java program that solves a **basic practice question** using variable assignment, output, and arithmetic operations.

It is intended for beginners to strengthen their understanding of **core Java fundamentals** through simple problem-solving.

---

## 📌 Program Overview

The program in this folder covers the following practice question:

- Calculate the **volume of a sphere** based on a predefined radius.

The program uses a predefined radius value and displays the calculated volume in the console, simultaneously demonstrating integer division behavior in Java.

---

## 🧪 Code Functionality

The program demonstrates:

### Arithmetic Operations
- Multiplication (`*`) and Division (`/`)

### Mathematical Formulas
- Volume calculation logic used: Volume = (3/4) * π * r * r * r (using 3.141592 for π)

### Variable Handling
- Initializing double-precision floating-point variables (`double`) for the radius and volume
- Calculating the result and concatenating it within a standard output print statement

The program is written in a **simple and readable format**.
