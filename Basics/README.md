# Java Fundamentals Practice

A dedicated repository tracking my daily progress in mastering core Java programming. This collection focuses on foundational concepts, syntax, and basic problem-solving to build a strong base for advanced topics and backend development.

## 🚀 Learning Objective

The primary goal of this repository is to solidify my understanding of Java fundamentals through hands-on coding. By writing, debugging, and executing small, focused programs daily, I am reinforcing my knowledge of variables, operators, input/output operations, and data type manipulation. 

## 🧠 Concepts Covered

This module (the `Basics` package) covers the following core Java concepts:
*   **Console Output:** Utilizing `System.out.print` and `System.out.println` for text and numeric rendering[cite: 4, 5].
*   **Variable Management:** Declaring, initializing, and continuously updating variables[cite: 12].
*   **Operators:** 
    *   Basic Arithmetic (`+`, `-`, `*`, `/`)[cite: 3].
    *   Unary Operators (Post-increment, Pre-increment, Post-decrement, Pre-decrement)[cite: 11].
*   **User Input:** Capturing dynamic runtime data using the `java.util.Scanner` class[cite: 2, 6, 7, 8].
*   **Type Casting:** Implementing both implicit (widening) and explicit (narrowing) type conversions between characters and integers[cite: 10].
*   **Mathematical Modeling:** Translating mathematical and geometric formulas into functional code[cite: 1, 9, 13].

## 📂 File Index

Below is a breakdown of the programs included in this practice set:

### Standard I/O & Variables
*   `Program1.java`: Demonstrates basic string output and formatting[cite: 4].
*   `Program2.java`: Highlights the difference between string concatenation and arithmetic addition in print statements[cite: 5].
*   `Variables.java`: Shows variable declaration, assignment, and sequential mathematical updates[cite: 12].
*   `Arithmetic_Operation.java`: Tests basic mathematical operators on double values[cite: 3].
*   `Urany_operator.java`: Explores the nuanced differences between `++x`, `x++`, `--y`, and `y--`[cite: 11].
*   `Type_CAsting.java`: Demonstrates how Java handles implicit character-to-integer conversion and forced integer-to-character casting[cite: 10].

### Mathematical Computations (Hardcoded Variables)
*   `Area_of_Circle.java`: Calculates the area using a fixed radius and the Pi constant[cite: 1].
*   `Surface_Area_of_Cuboid.java`: Computes surface area based on fixed length, breadth, and height[cite: 9].
*   `Volume_of_Sphere.java`: Calculates the volume of a sphere using standard geometric formulas. 

### Dynamic Computations (Scanner Input)
*   `Area_of_Circle_via_input.java`: Calculates the area of a circle based on a user-provided radius[cite: 2].
*   `Simple_interest.java`: Takes principal, rate, and time as inputs to generate simple interest and the total amount[cite: 6].
*   `square_of_a_number.java`: Prompts the user for an integer and returns its square[cite: 7].
*   `Sum_of_numbers.java`: Reads three separate integers from the user and outputs their sum[cite: 8].
