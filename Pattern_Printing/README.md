# Java Pattern Printing & Nested Loops

A dedicated repository section tracking my daily practice in mastering Java's nested looping mechanisms. This collection focuses on implementing 2D iterative logic to generate visual patterns, shapes, and grids to build a strong foundation for matrix operations and advanced algorithm development.

## 🚀 Overview
This module covers the practical application of nested `for` loops in Java. The programs demonstrate how to control outer and inner execution cycles to process coordinate-based grid systems, allowing for the dynamic printing of character, number, and symbol patterns based on user input.

## 🛠️ Tech Stack & Tools
* **Language:** Java
* **Core Concepts:** Nested `for` Loops, 2D Grid Logic, Standard I/O, Explicit Type Casting (ASCII)

## 📂 File Index & Structure

### Symbol & Shape Patterns
* `StarRectangle.java`: Uses nested loops to generate a customizable rectangle of stars based on separate row and column inputs from the user.
* `StarSquare.java`: Generates a symmetrical NxN square grid of stars, demonstrating equal boundary conditions for both inner and outer loops.

### Numerical & Alphabetical Grids
* `NumberSquare.java`: Implements coordinate-based printing using the inner loop variable to output a dynamic NxN square grid of repeating sequential numbers.
* `AlphabetSquare.java`: Utilizes explicit type casting `(char)(j+64)` within a nested loop to map integer coordinates to their corresponding uppercase English alphabets in a square grid format.

## 💡 Key Learnings & Features
* **Nested Iteration:** Mastering the relationship between outer loops (handling rows) and inner loops (handling columns) to efficiently traverse and manipulate 2D space.
* **Coordinate Mapping:** Understanding how to use loop index variables (like `j`) as mathematical operators and ASCII cast targets to generate alphabetical and numerical layouts purely through logic.
* **Dynamic Grid Control:** Using the `Scanner` class to establish the termination conditions (`n`, `rows`, `columns`) of multi-layered loops dynamically at runtime.
