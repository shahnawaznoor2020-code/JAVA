# Java Core Fundamentals & DSA Practice

A dedicated repository tracking my daily practice in mastering Java's core programming constructs. This collection focuses on foundational concepts, conditional logic, and looping mechanisms to build a strong foundation for algorithmic problem-solving and backend development.

## 🚀 Overview
This repository encompasses practical applications of basic Java syntax, control flow, and iterative sequences. The programs demonstrate how to manipulate variables, control execution cycles, process dynamic user inputs, and implement mathematical modeling to solve introductory computational challenges.

## 🛠️ Tech Stack & Tools
* **Language:** Java
* **Core Concepts:** Variables & Data Types, Standard I/O, Relational & Logical Operators, Control Flow (`if-else`, ternary), `for` Loops, Mathematical Progressions

## 📂 File Index & Structure

### Basics: Variables & Standard I/O
* `Program1.java` & `Program2.java`: Demonstrates fundamental string output and concatenation.
* `Variables.java` & `Arithmetic_Operation.java`: Covers variable declaration and basic arithmetic on numeric types.
* `Type_CAsting.java` & `Urany_operator.java`: Explores implicit/explicit casting and prefix/postfix operators.
* `Area_of_Circle_via_input.java`, `Simple_interest.java`, `square_of_a_number.java`, `Sum_of_numbers.java`: Implements the `Scanner` class to compute dynamic mathematical operations.
* `Area_of_Circle.java`, `Surface_Area_of_Cuboid.java`, `Volume_of_Sphere.java`: Applies hardcoded variables to geometric formulas.

### Control Statements & Conditional Logic
* `Relational_operator.java`: Demonstrates foundational relational operators.
* `Non_negative.java`, `Even_Odd_Number.java`, `Absolut_value.java`: Utilizes basic `if-else` blocks and modulo arithmetic.
* `Divisibility_By_5.java`, `divisibility_by_3_or_5.java`, `Divisibility_By_5_or_3.java`: Implements logical AND/OR operators for divisibility filtering.
* `If_4_digit_number.java`, `_69_Integer_Checking.java`, `Integer_checking.java`: Validates number ranges and fractions.
* `Profit_Loss.java`, `Profit_Loss_percentage.java`, `Rectagle.java`, `Quadrant.java`, `Sides_of_triangle.java`: Applies nested conditional logic to real-world financial, geometric, and mathematical evaluations.
* `GreatestOfThree.java`, `GreatestOfThreeTernaryOperator.java`, `EvenOddTernaryOperator.java`, `TernaryInVariable.java`: Showcases both traditional and concise ternary (`? :`) decision-making paths.

### Iteration & Loop Constructs
* `BasicLoop.java`, `PrintingNTimes.java`, `Print1To100.java`, `PrintNumberFromNTo1.java`: Covers standard `for` loops and bidirectional counting.
* `TableOf17.java`, `TableOfAnyNumber.java`: Generates hardcoded and dynamic multiplication tables.
* `AllEvenNumbersBetween1To100.java`, `AllOddNumberDivisibleBy3Between1To100.java`: Uses modulo conditions and optimized step increments to filter sequences.
* `ArthemeticProgression2_5_8.java`, `AnyArthematicProgression.java`, `PrintSeries99_95_91TillTheLastPositiveNumber.java`: Generates dynamic Arithmetic Progressions (AP).
* `GeometricProgression.java`, `AnyGeometricProgression.java`: Generates Exponential/Geometric Progressions (GP).
* `SequencePrinting.java`: Implements dual-variable tracking within a single loop execution.
* `AllAlphabetWithTheirASCIIValue.java`: Uses explicit type casting within an iteration to print ASCII maps.

## 💡 Key Learnings & Features
* **Algorithmic Validations:** Applying mathematical theorems (like Triangle Inequality) and financial formulas to solve practical problems strictly through code logic.
* **Decision Making & Compound Logic:** Routing program logic effectively using structured conditional blocks, `else-if` ladders, and the Ternary Operator (`? :`).
* **Loop Optimization:** Understanding that adjusting loop update statements (e.g., `i = i + 2`) can be significantly more efficient than evaluating conditions inside the loop block.
* **Dynamic Sequences:** Applying progression formulas to generate both forward and bidirectional mathematical sequences dynamically.
