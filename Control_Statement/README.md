## đź§  Concepts Covered

This module (the `Control_Statement` package) covers the following core Java concepts:
* **Relational Operators:** Utilizing `<`, `>`, `==`, `<=`, `>=`, and `!=` for boolean evaluations.
* **Conditional Statements:** Implementing `if`, `if-else`, and `else-if` ladders to control program execution flow.
* **Logical Operators:** Combining conditions using AND (`&&`) and OR (`||`) for complex decision-making.
* **Ternary Operator:** Writing concise, inline conditional assignments using the `? :` syntax.
* **Algorithmic Logic:** Translating mathematical constraints, divisibility rules, and financial formulas into functional code.

## đź“‚ File Index

Below is a breakdown of the programs included in this practice set:

### Basic Conditionals & Relational Operators
* `Relational_operator.java`: Demonstrates the foundational relational operators and how they evaluate to boolean values.
* `Non_negative.java`: Utilizes a basic `if` statement to verify if a user-provided number is greater than or equal to zero.
* `Even_Odd_Number.java`: Applies the modulo operator to determine if a number is even or odd.
* `Absolut_value.java`: Computes and outputs the mathematical absolute value of a given integer.

### Logical Operators & Divisibility
* `Divisibility_By_5.java`: Checks if a positive integer is evenly divisible by 5.
* `divisibility_by_3_or_5.java`: Implements the logical OR operator to check if a number is divisible by either 3 or 5.
* `Divisibility_By_5_or_3.java`: Uses an `if-else if` ladder and the logical AND operator to categorize a number's divisibility by 3, 5, or both.
* `If_4_digit_number.java`: Validates if a user's input falls strictly within the four-digit range.
* `_69_Integer_Checking.java`: Evaluates the magnitude of a number against absolute boundaries using logical conditions.
* `Integer_checking.java`: Compares a double value against its type-casted integer counterpart to check for fractional parts.

### Nested Logic & Ternary Operations
* `GreatestOfThree.java`: Uses nested `if-else` statements to find the maximum of three numbers.
* `EvenOddTernaryOperator.java`: Replaces traditional `if-else` blocks with a ternary operator to check for even or odd numbers.
* `TernaryInVariable.java`: Demonstrates using the ternary operator to conditionally assign a value to a variable.
* `GreatestOfThreeTernaryOperator.java`: Implements a nested ternary operator to determine the greatest of three numbers in a single line.

### Mathematical, Financial & Geometric Validations
* `Profit_Loss.java`: Compares cost price and selling price to calculate the exact profit or loss amount.
* `Profit_Loss_percentage.java`: Expands on financial logic to compute the exact percentage of profit or loss.
* `Rectagle.java`: Calculates and compares the area and perimeter of a rectangle using conditional logic.
* `Quadrant.java`: Evaluates X and Y coordinates to determine the exact Cartesian quadrant or axis.
